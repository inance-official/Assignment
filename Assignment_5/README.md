# Question

- 이 프로젝트는 파일 쓰기,읽기 속도를 측정하는 프로그램

- 다음과 같은 네가지 case의 각 각 속도를 측정한다.
  - 파일을 여닫는 것은 측정하지 않고, fwrite 작동만 속도를 측정한다.
  - 파일을 여닫는 것은 측정하지 않고, fread 작동만 속도를 측정한다.
  - 파일을 여닫는 것을 포함하여, fwrite 작동 속도를 측정한다.  
  - 파일을 여닫는 것은 포함하여, fwrite 작동 속도를 측정한다.
- 프로그램 실행 시, 파일 경로와 반복횟수를 지정해준다.
- 파일에 쓰여질 배열은 1024*1024 (1MB) 이며, 원소 하나의 크기는 char의 크기인 1이다.
- 파일에 저장할 배열의 원소 개수는 가장 처음은 1024개 이며, 이후 10240개 씩 증가하며 실행한다.
- 파일 읽기의 경우도 위 조건과 동일하다.
- 반복 횟 수 만큼 쓰기나 읽기 작업을 반복한다.
